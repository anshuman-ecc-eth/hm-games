# Games Collection

This repository contains modified browser games built with [crisp-game-lib](https://github.com/abagames/crisp-game-lib).

## Changes Made

### REBIRTH (`games/rebirth.html`)
- Updated scrolling text from "here's a rather crude simulation..   " to "press to jump. collect yellow diamonds. obstacles fling you to the other side... "

### SQUARE BAR (`games/squarebar.html`)
- Replaced static multi-line text with a scrolling marquee inside the gray box
- Text: "long press. collect yellows. avoid reds."
- Characters scroll from right to left within the box boundaries (x=30 to x=68)
- Text disappears slightly before the left inner edge for a cleaner effect
